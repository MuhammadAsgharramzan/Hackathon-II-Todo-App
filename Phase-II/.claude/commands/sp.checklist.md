# Generate Custom Checklists

This command generates custom checklists for various project activities based on the current context and requirements.

## Usage

```bash
claude explain "create a deployment checklist"
```

## Purpose

This command creates tailored checklists for:
- Code reviews
- Testing procedures
- Deployment processes
- Security audits
- Quality assurance
