# Document Architectural Decisions

This command helps document architectural decisions following the Architecture Decision Record (ADR) format.

## Usage

```bash
claude explain "document the decision to use a multi-phase approach for the todo app"
```

## Purpose

This command creates structured documentation for important architectural decisions made during the project, including:
- The context and problem statement
- The decision made
- The rationale behind the decision
- The consequences of the decision
