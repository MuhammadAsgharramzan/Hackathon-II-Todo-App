# Create Project Constitution

This command manages the project constitution, which defines the fundamental principles and rules governing the project.

## Usage

```bash
claude explain "update the project constitution with multi-phase development principles"
```

## Purpose

This command maintains the project constitution that establishes:
- Core principles and values
- Development guidelines
- Quality standards
- Governance rules
- Decision-making processes
