# Cross-Artifact Consistency Checks

This command performs consistency analysis across project artifacts to ensure alignment between specifications, implementations, and documentation.

## Usage

```bash
claude explain "check consistency across all phases of the todo app"
```

## Purpose

This command verifies that:
- Specifications match implementations
- Documentation aligns with actual behavior
- Configuration files are consistent with architecture
- Dependencies are properly maintained
