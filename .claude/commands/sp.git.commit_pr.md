# Commit and Create Pull Requests

This command automates the process of committing changes and creating pull requests following project conventions.

## Usage

```bash
claude explain "commit changes and create a pull request"
```

## Purpose

This command handles:
- Staging appropriate files
- Creating well-formatted commit messages
- Pushing branches
- Creating descriptive pull requests
- Following conventional commit formats
